# Smart-Industrial-Safety-Monitoring-System
An AI-powered workplace safety monitoring system developed during my engineering internship to enhance the workers safety on construction site through real-time PPE detection, automated alerts, and incident documentation.

## Project Overview
Construction sites often face challenges in ensuring that all workers consistently wear Personal Protective Equipment (PPE). Manual monitoring is time-consuming and cannot continuously observe every worker, which may delay the detection of safety violations and the response from workplace safety managers.
To address this issue, this project utilizes Artificial Intelligence and Computer Vision technologies to analyze live video streams from CCTV(RSTP) cameras, automatically detect PPE violations, and notify workplace safety managers in real time. By enabling continuous monitoring and rapid response, the system enhances workplace safety and supports more efficient safety management practices.

## Objectives
The project was developed with the following objectives:
- Detect PPE violations in real time using Artificial Intelligence and Computer Vision.
- Reduce reliance on manual workplace safety inspections by automating PPE compliance monitoring.
- Provide immediate notifications when PPE violations are detected.
- Support workplace safety managers with continuous monitoring through a web-based dashboard accessible on both desktop and mobile devices.
- Improve workplace safety by enabling timely intervention through automated monitoring and real-time alerts.
