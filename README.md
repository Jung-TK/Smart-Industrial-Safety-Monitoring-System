# Smart-Industrial-Safety-Monitoring-System
An AI-powered workplace safety monitoring system developed during my engineering internship to enhance the workers safety on construction site through real-time PPE detection, automated alerts, and incident documentation.
---
## Project Overview
Construction sites often face challenges in ensuring that all workers consistently wear Personal Protective Equipment (PPE). Manual monitoring is time-consuming and may not provide immediate awareness or response from workplace safety managers when safety violations occur.
To address this issue, this project utilizes Artificial Intelligence and Computer Vision technologies to automatically detect PPE violations and notify workplace safety managers in real time. By enabling continuous monitoring and rapid response, the system enhances workplace safety and supports more efficient safety management practices.
